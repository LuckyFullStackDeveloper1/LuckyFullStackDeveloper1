Detailed Solution

Download Solution PDF

The correct answer is Nexus.

Key Points

  • The first web browser or browser editor rather - was called WorldWideWeb as, after all, when it was written in 1990 it was the only way to see the web.
  • Much later it was renamed Nexus in order to save confusion between the program and the abstract information space (which is now spelled World Wide Web with spaces)

Additional InformationInternet Explorer:

  • Internet Explorer is a series of graphical web browsers developed by Microsoft and included in the Microsoft Windows line of operating systems, starting in 1995. It was first released as part of the add-on package Plus! for Windows 95 that year.

Mosaic:

  • Mosaic is a web browser.
  • In 1993, the “Mosaic” web browser was released. 
  • It was developed by the National Centre for Supercomputing Applications (NCSA).
  • It had the feature of adding images and an innovative graphical interface.
  • It was “the world’s first popular browser”.

Mozilla:

  • Mozilla Firefox is a free and open-source web browser developed by Mozilla Foundation and released in September 2002. It is compatible with many operating systems like Windows, Linux, Mac, etc.
  • In addition, Ctrl + Tab combination key is used to switch between different tabs of the browser and can be used in all types of browsers.
